# IronTrack Fitness

A premium fitness tracking web application featuring:
- **Workout Logging**: Track sets, reps, and exercises.
- **Attendance Tracking**: Monitor gym time with manual check-in/out and QR scanning.
- **Progress Tracking**: Log weight and view daily streaks.
- **Nutrition Targets**: Personalized protein and water goals based on profile.
- **Progress Photos**: Save and view transformation photos.

## Setup

1. Open `index.html` in a browser or host it on a static server.
2. (Optional) Connect to a Supabase backend for cloud sync by creating a project at [supabase.com](https://supabase.com) and updating `app.js` with your keys.

## Deployment

This is a static web app and can be hosted for free on specific services like Render, Netlify, or Vercel.
